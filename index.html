Tomoko

Java library REST PATCH Json RFC 6902


View code
GitHub pages

Tomoko is a Java 8 library that allows developers to implement REST PATCH calls in accordance with JSON Patch (RFC 6902) standard.

As a developer you just have to attach handlers (lambdas) for each valid path. These handlers will receive as arguments values of all defined path parameters and the parsed operation value in case of the ADD operation.

Tomoko takes care of parsing the whole request, checking if the request is valid according to the defined model, finding handlers and executing them.

It supports Jackson and Gson libraries so you don't have to parse JSON Patch requests yourself. However valid requests can still be created without using JSON at all which is very useful for testing purposes.
